# רנבוק בדיקה ידנית למייסד - 30 דקות לפני שליחה לטסטרים

מטרה: לבצע בדיקה מהירה ומעשית של Family App לפני שליחה ל-3 טסטרים ראשונים. הבדיקה צריכה להתבצע כאילו אתם משתמש משפחתי רגיל, בלי הנחות מוקדמות ובלי לדלג על מסכים.

משך מומלץ: 30 דקות.

ציוד נדרש:

- מכשיר ראשון או דפדפן ראשון.
- מכשיר שני או דפדפן אחר / חלון פרטי.
- אפשרות להקליט קול.
- אפשרות לקבל התראות.
- שעון למדידת תחושת מהירות.

## 1. בדיקת איפוס נקי

פתחו:

`/app.html?reset=1`

בדיקות:

- [ ] המסך נטען בלי שגיאה.
- [ ] אין קבוצה קיימת.
- [ ] אין שם משתמש קודם.
- [ ] אין משימות ישנות.
- [ ] אין הזמנות או קודים ישנים.
- [ ] ברור מה הפעולה הראשונה שצריך לעשות.

תוצאה:

- [ ] עבר
- [ ] אזהרה
- [ ] נכשל

## 2. יצירת קבוצה עם שם חופשי ותפקיד

במכשיר הראשון:

1. התחילו כמשתמש חדש.
2. הזינו שם תצוגה חופשי בעברית.
3. בחרו או הזינו תפקיד משפחתי.
4. צרו קבוצה חדשה.

בדיקות:

- [ ] אפשר להזין שם חופשי בלי מגבלה מוזרה.
- [ ] שם בעברית נשמר ומוצג נכון.
- [ ] התפקיד נשמר ומוצג נכון.
- [ ] אין צורך להבין מושגים טכניים כדי ליצור קבוצה.
- [ ] אחרי יצירת הקבוצה ברור שהמשתמש נמצא בתוך קבוצה.

תוצאה:

- [ ] נכשל
